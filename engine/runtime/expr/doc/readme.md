# Expression Language

## Literals

- integer: `42`
- float: `3.14`
- string: `"hello"`
- bool: `true`, `false`
- null: `null`
- array: `[1, "a", true]`

## Operators

### Precedence (high to low)

1. `[key]` `.method()`
2. unary `!` `-`
3. `*` `/`
4. `+` `-`
5. `<` `<=` `>` `>=`
6. `==` `!=` `in`
7. `&&`
8. `||`
