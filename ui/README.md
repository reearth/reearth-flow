# Re:Earth Flow Web
Frontend to build and setup workflows to calculate and convert various types of data through an easy to use UI.

## Development

### Main libraries & services
- React 18
- TypeScript 5
- Vite 5
- Storybook 8
- Yarn 4

### Install prerequisites

```console
yarn

yarn start
// or
yarn storybook
```
